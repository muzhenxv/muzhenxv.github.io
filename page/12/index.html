最近在做物体识别方面的工作，整体流程是接收直播流，用yolo进行特定物体识别，推送处理后的视频流到指定地址。其中，需要用到ffmpeg进行处理。现将ffmpeg使用过程总结一下。

install

sudo apt-get install ffmpeg

process

编码格式

可以使用以上命令去查看文件的视频和音频编码格式

视频切割：

-i 表示待处理的视频源

-ss，-t 表示切割视频的起始时间点和切割时长。有两种写法：0，00：01：20。前者表示秒，后者是时:分:秒格式。

-to 表示结束时间点。 但如果 -ss 放在 -i 前面，则效果和 -t 一样。有的blog在这个问题上说反了，特此说明。只要 -ss 始终放 -i 后面，就不会出问题了。

-y 表示如果输出文件存在，则覆盖原文件

-c 表示编码器，是-codec的缩写。 -c copy指复制原视频编码流。

-c:v -c:a 表示分别指定视频音频编码流格式。也可以写成 -vcodec -acodec或者 -c:0 -c:1一般0是视频流，1是音频流。如果出现多音频流，可以用-c:a:0形式去选择。

-vn -an 分别表示屏蔽视频流，音频流。

在使用命令时好像必须同时指定视频流和音频流的处理，不能只指定某一个。

《打开文学的方式》

《打开文学的方式》——王敦

1.1 “文学名著看不进去，一定是我打开的方式不对”

众口难调，感动你的不一定能感动他或她，若去追究和讨论“什么”让我感动，势必会忽略我是“怎样”被感动，以及“为什么”我被感动了——对这后两者的分析，才是“解读”，才能有助于打开我们所遭受的感动，看看它到底是怎么回事儿。

1.2 学会“打开文学的方式”，是一件正经事

“解读”之事，说到底，绝不是去“还原”作者的“圣旨”，也不是选择一位权威的“批评家”，然后站到他的身后，成为一个无关紧要的“零”，而是充分发挥读者的“主体”作用，为作品有效地生成新的意义。解读出来的新的意义，是解读者的私有财产，并参与到与作者，与其他读者的交流之中，成为读者自己对文学这件事儿的贡献。……“文学解读讲义”所讲的就是读者如何发挥主体性进行“二度创造”。

想一想美国电影《黑客帝国》和《楚门的世界》吧。我们被别人精心炮制的说辞和字符矩阵所包围，如果不去“close reading”，多分析分析“为什么”（why）和“如何”（how）的问题，被其里面的“what”所裹挟、忽悠、催眠，我们被骗的几率就是99.9%了。在社会上往往是整整一代代的人集体被骗。（很显然他们都没有读过这本书，幽默一下儿。）在比较糟糕的时代，比如希特勒时代的德国，你个人幸福追求的回旋余地就变得近乎于零了。

1.3 七次Lecture帮你打开文学

我们讨论文学的解读，绝不仅仅是为了欣赏文学本身，更重要的是学会从语言符号里来解读社会话语的能力，学会社会性思考和生存。

book review

《进化！进化？达尔文背后的战争》

01 一个严重的问题：我们从何而来

休谟指责设计论说……无疑这个上帝是一个低能儿。（P8）
休谟对于自然神学的批判是怎样的？[TBC]

科学界迫切的任务是，需要彻底排除超自然的力量，向自然寻求真正的规律，以科学的态度研究生物和人的由来。（P14）

应该先明确何为科学。然后再看科学界迫切的任务是什么？科学是否需要排除超自然的力量，不能兼容？研究生物和人的由来难道成了整个科学界的任务？

02 拉马克主义，以及铁匠的儿子和老鼠的尾巴

根据自己的这一观点，出于学术上争论的目的，居维叶对提出了进化论的老同事拉马克竭尽嘲笑讽刺之能事。（P14）

居维叶何以对拉马克的观点和证据视而不见？如果没有看到，是何缘故？如果看到了，又是怎样去作出辩驳和批评的呢？

“用进废退”“获得性遗传”（P15）

退不是指退步，向不好的方向发展。只是指因为官能派不上用场而退化这一中性的事实性描述而已。

“获得性遗传”并没有能在书中被严肃的解释。

需要考虑遗传退火的问题。或许不是用进废退。而是持续不断的基因变化以及更适合环境的基因在竞争中获胜，从而造成用进废退的表象。应该搞清楚何为因何为果。

他认为物种只是一个认为的概念，事实上并不存在。（P16）

这是一个概念厘定问题。还是有必要深思一下的。必须搞清楚在怎样的问题下应当关注怎样的概念，才不至于落入无意义的逻辑语辞游戏，也不至于忽略真正的细节和逻辑关键。

如果有喜欢人文学科的朋友们加入到这个讨论中来的话，无疑会把这个讨论搅得一团糟……而纯正的进化理论，必须把这种力量排除在外，因为对动物的“意志”的依赖，一不留意，就会滑向“神的意志”，有变为自然神学的危险。（P18）

“纯正的进化理论”？谁来定义纯正？前文就以出现这样的问题，作者无任何论证的将科学和神学、人文主观进行想当然的割裂，好像这是一个不言自明的真理与共识一样。这样的态度是非常危险的，尤其作为科普读物。既然是科普，那么不管是个人私货还是未经定论的事情，甚至于哪怕相对定论的事情，都应当清晰的论述清楚它的前设，推理，结论并其存在的漏洞以及未竟之处。尤其是前设和缺陷最为重要。如若不然，不过是普及了知识，却愚弄了思想。

人文主观的力量或许并没有那么小，不仅是潜藏在社会诸领域之下的巨龙，足以影响到人类所以为的客观领域也未可知。

自然神学从本书一开始就已经成为了被批评的靶子，然而仅仅只是被零散的论述，而且对于其死而不僵甚至于重新活跃到目前为止并没有论述。

关于“科学”问题，有必要回顾波普尔的科学哲学。如有精力，日后补足。[TBC]

但由于受到了居维叶的打压，所以在当时的影响较小。（P19）

读到本章结束，我的感觉是虽然达尔文及达尔文主义者对拉马克之思想也有嘲笑讽刺但止于学术，而居维叶则还进行了人身攻击，有学品人品皆欠佳之嫌。这是作者言辞的误导还是确实如此呢？

达尔文没有度过拉马克的原著……（P19）

没读过原著，却对拉马克有各种批评和赞美……这治学的态度可不严谨。或许是当时难以搞到原著资料？

紫日

紫日——论文艺创作之前提

2013年2月3日星期日

《紫日》这部片子就细节而言颇有可圈可点之处。

战火纷飞的年代，却能有一片静谧了无人烟的大兴安岭，这是一种幸运？白种人，黄种人，日本人，中国人，俄国人，不同的肤色，不同的立场，曾经有刻骨铭心的民族大恨，曾经有忠诚爱国的思想灌输，却在荒原之中为了生存主动将生命连结在一起。大兴安岭给出了一个苍茫天地下的人生战场，在这里，三个人进行的是一场源自内心的战争，他们的行为反映出来的是人最原初的追求。但这样的追求，这样的大兴安岭是无法长居的，终究要出去，而且是从何处来便往何处去，回归到那个充满人烟的战场，回归到那个需要去捡拾民族捡拾家国责任的现实世界。

另一方面，哪怕是在大兴安岭之中，依旧有着虎豹虫豺的危险，依旧需要拿起刀枪。净土又在何方？

日本女子的态度变化，中国农民数次将刀垂落入土里的细节变化，都可以去深入挖掘，但这些特殊的寓意这些精妙的设计并不足以使这部电影成为一部佳作。原因很简单——无法吸引观者！

电影一旦被搬上了荧幕，就与创作者失去了关系。再好的立意，再好的构思，都不属于观者，也就自然无法打动观者。同样的，一部电影对于导演本人而言再次再没有价值但如果能够凭借它的无厘头它的各种搞怪去吸引观者，并让观者自觉地从中挖出某些深意，那便是一部佳作。譬如《让子弹飞》，姜文的初始立意早已不重要，就算这是一部哗众取宠之作又如何呢？它成功了吸引了观者，并衍生了千百种的解读，追连出许多永恒的话题。借用一句老话，一千个读者便有一千个哈姆雷特。但是想要出现一千个哈姆雷特的前提乃是有一千个人愿意去读它愿意去深究它，否则不过一场空谈，束之高阁的货物罢了！

故而，文艺创作如果是以公众为对象，就不该有太过强烈的个人观念，此处之意不在于让创作者放弃自身的思想而去迎合大众潮流，而在于创作者需要铭记以公众为对象就必须以吸引公众为第一前提！再次前提下，创作者可以尽情展示其本人的思想，去改变这个世界！

《紫日》的失败就在于没有做到这第一前提，相对于两个小时的电影片长有太多的絮叨啰嗦，太少的精妙激动，让观者昏昏然欲睡。

TF-IDF

TF是指特定词在包含其的文件中出现的频率。

IDF是指包含该词的文件占全部文件数的比例的倒数的对数。

tf*idf就是词重要性度量。但是一般可以按行（样例）进行正则化，将tf*idf规范到0-1,并且这样子相同词在不同样例中的数值（重要性）也就不再一样。等于是进一步考虑到了整个句子结构的特性。
