idf 计算技巧

假设不同词共有N个，文档共有M个，单文档最大长度是L。

我们看如下两种计算方式。

第一种是先作出所有词的词典，然后对每个词进行遍历，看它在多少个文档中出现，复杂度是N×M：

第二种是遍历所有文档，记录每个文档中出现的词，复杂度是M×L：

显然，两者效率相差极大。一般L不过在10左右，而N却可以高达上万。
