PR

使用别人的repo代码，发现问题想要更正以后合并，可以通过以下方法：

  1. fork repo
  2. git clone自己fork过来的repo到本地
  3. 本地正常修改提交
  4. 在自己fork的repo上发pr申请，等待原仓库管理员merge

本地仓库和线上仓库关联

想要把本地仓库和一个线上的仓库合并：

  1. 本地仓库和线上仓库同名

  2. 如果本地仓库只是文件夹，需要git init

  3. 创建关联，git remote add origin <线上repo ssh地址>

  4. 先把线上仓库拉取合并下来，git pull origin master

  5. 本地仓库推到线上，
